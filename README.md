e380_kk_kernel
==============

Acer Liquid E380 kernel source

Tested with E380 dual sim PLUS version

Usage:

change scripts build.sh and clean.sh to point to your toolchain

to clean: ./clean.sh

to build:

./build.sh
cd kernel-assembly
./assemble.sh

- push testkernel.zip to device and flash with cwm recovery
- have fun!
